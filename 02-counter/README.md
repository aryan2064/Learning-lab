# Counter

A numeric counter with increase, decrease, and reset controls.

## Features

- Three actions: increment, decrement, reset
- Number color reflects positive/negative/zero state
- Shared button logic using class-based branching

## Run Locally

1. Open this folder.
2. Launch index.html in your browser (or with Live Server).

## Main Files

- index.html
- styles.css
- app.js
