# Learning Lab

A collection of small, focused JavaScript projects built with HTML, CSS, and vanilla JS.

## Quick Start

1. Clone this repository.
2. Open any project folder.
3. Run the index.html file directly in a browser, or use a local static server.

Example with VS Code Live Server:

- Open a project folder (for example, 01-color-flipper)
- Start Live Server on index.html

## Tech Stack

- HTML5
- CSS3
- JavaScript (ES6, no framework)
